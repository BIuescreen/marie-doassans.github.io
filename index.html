VulnWatch

Agrégateur de veille cybersécurité avec alertes automatisées. Collecte toutes les heures les nouvelles CVE depuis NVD, CERT-FR et Exploit-DB, les filtre selon des règles personnalisées (seuil CVSS, mots-clés) et envoie des alertes Discord ou Email via des workflows n8n self-hosted.

Python 3.11 Flask SQLAlchemy MySQL 8 Docker n8n discord.py NVD API CERT-FR RSS TailwindCSS Alembic pytest
Période Novembre 2025 → Avril 2026
Sources CVE NVD API · CERT-FR RSS · Exploit-DB scraping
Alertes Discord DM (embed coloré) · Email HTML via n8n
Sécurité PBKDF2-SHA256 · AES-256 · CSP nonce · itsdangerous
Tests 47 tests pytest + unittest.mock
Déploiement VPS Ubuntu · Nginx · Gunicorn · Let's Encrypt

Cyberhoot

Plateforme web de sensibilisation à la cybersécurité par le jeu. Quiz thématiques interactifs avec système de badges et gamification, trois rôles distincts (joueur, rédacteur, administrateur), validation VirusTotal des fichiers uploadés et tableau de bord statistiques pour les créateurs.

Python 3.x Flask SQLAlchemy Werkzeug MySQL Docker VirusTotal API Alembic Jinja2 TailwindCSS Flask-Mail
Période Novembre 2025 → Avril 2026
Rôles Joueur · Rédacteur · Administrateur
Fonctions Quiz CRUD · Badges · Scores · Notifications email
Sécurité PBKDF2 · CSP nonce · tokens itsdangerous
